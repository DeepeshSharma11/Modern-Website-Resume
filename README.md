# Deepesh Sharma - Portfolio & Resume

A modern, responsive, and performance-optimized single-page portfolio website showcasing my experience, academic background, technical skills, and projects.

## Tech Stack & Features

- **Styling**: Tailwind CSS for a modern, utility-first responsive layout.
- **Interactions & Graphics**: Three.js for interactive particle backgrounds, Lucide icons, and light CSS reveal animations.
- **SEO & Schema**: OpenGraph metadata, custom canonical URLs, XML schemas, and JSON-LD Person schema optimization.
- **Performance**: Optimized rendering with layout containment, mobile-responsive particle disabling, and scroll-bound animation throttling.

## Project Structure

- `index.html` - Page structure and SEO/metadata configurations.
- `css/styles.css` - Custom style layers, variables, and typography overrides.
- `js/main.js` - Section routing, mobile menu logic, scroll-reveal transitions, and Three.js canvas management.
- `Assets/` - Shared media, SEO assets, site icons, and downloadable PDF resume.

## Local Setup

To run the project locally, clone the repository and open `index.html` in any modern web browser. Alternatively, serve it using a lightweight development server:

```bash
# Using Python's built-in HTTP server
python -m http.server 8000

# Or using Node's serve utility
npx serve .
```

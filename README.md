# Deepesh Sharma Portfolio

Static portfolio and resume website.

## Structure

- `index.html` - page markup
- `css/styles.css` - custom styling
- `js/main.js` - navigation, reveal animations, and background particles
- `Assets/` - favicon and resume PDF

## Run

Open `index.html` in a browser or deploy the folder to any static host.
